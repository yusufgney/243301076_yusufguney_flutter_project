# 🎬 Castflow

Castflow is a modern casting management platform built with Flutter and Firebase, designed to bridge the gap between actors and casting agencies.

## ✨ Features

- **Double-Sided Dashboard:** Tailored experiences for both **Actors** and **Agencies**.
- **Modern UI/UX:** Clean, professional interface with **AMOLED Dark Mode** support.
- **Project Management:** Agencies can create and manage casting calls easily.
- **Real-time Applications:** Actors can browse casting calls and apply instantly.
- **Persistent Themes:** User-configurable dark/light mode that persists across sessions.
- **Robust Tech Stack:** State management with **Riverpod** and persistence with **SharedPreferences**.

## 🛠️ Tech Stack

- **Framework:** [Flutter](https://flutter.dev) (Material 3)
- **State Management:** [Riverpod](https://riverpod.dev)
- **Database/Auth:** [Firebase](https://firebase.google.com)
- **Routing:** [go_router](https://pub.dev/packages/go_router)
- **Local Storage:** [shared_preferences](https://pub.dev/packages/shared_preferences)

## 🚀 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/castflow.git
    ```
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the app:**
    ```bash
    flutter run
    ```
